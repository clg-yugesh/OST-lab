# OST-lab

Open Source Technology Practical course for II B.Sc Computer Science (Data Science and Analytics), Semester IV.

| Detail | Information |
|--------|-------------|
| **Course Code** | 24DCS410 P |
| **Semester** | IV |

| **Credits** | 2 |

| **CIA Marks** | 40 |
| **End Semester Marks** | 60 |

## Course Objective

Understand fundamentals of web design and gain knowledge in CSS and JavaScript. Build dynamic, interactive websites with server-side scripting using PHP and database connectivity with MySQL.

## Prerequisites

- Basic computer knowledge
- Understanding of HTML fundamentals
- Logical thinking and problem-solving
- No prior web development experience required

## Installation

```bash
# Clone repository
git clone https://github.com/Yugesh-003/OST-lab.git
cd OST-lab

# For JavaScript labs (just need a browser and text editor)
# Recommended: Visual Studio Code

# For PHP/MySQL labs
# Install XAMPP or similar stack (Apache, PHP, MySQL)
```

## Lab Exercises

### JavaScript Exercises

1. Operator and Expressions
2. Conditional statements
3. Iteration statements
4. Array Processing
5. Functions
6. Dialog box
7. Java script document object model
8. Event handling

### PHP Exercises

9. Program to create Forms in PHP
10. Program to implement Regular Expressions in PHP
11. Program to implement Include
12. Program to implement Require
13. Program to create Cookies in PHP
14. Program to create Sessions in PHP
15. Database – MySQL, Tables implementation in PHP


## Testing

**JavaScript**: Use browser console (F12 → Console tab)

**PHP/MySQL**: Use XAMPP/local server
```bash
# Start Apache and MySQL in XAMPP
# Access at http://localhost/OST-lab/
```
